# instagram-profile-picture-viewer
Simple webapp to see instagram profile pictures in bigger size if available.

Run:
- with Maven: mvn spring-boot:run

Technology stack:
- Spring Boot (MVC)
- Thymeleaf
- HTML
- Javascript & jQuery
- Bootstrap
- Jackson
